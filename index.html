ETIQUETAS SEMANTICAS ESTRUCTURALES - SON DE LAS MAS IMPORTANTES!

Div es una etiqueta contenedora que no tiene SEMANTICA
header - Cabecera de un sitio web o de una seccion
main - Define la seccion principal del documento, solo puede existir una etiqueta main por documento
footer - Pie de pagina de un sitio web o una seccion
nav - Representa una navegacion, menu, botonera, etc
article - Representa una seccion de autocontenido
section - representa una seccion de contenido generico, podria incluir varios articles o al reves
aside-Represnta un contenido secundario
adress - Representa una direccion de contacto!

Etiquetas de bloque vs etiquetas de linea

Etiqueta de bloque por excelencia - DIV, una etiqueta de bloque es aquella que genera salto de lineas con sus etiquetas hermanas
Etiqueta de linea por excelencia - span , una etiqueta de linea solo ocupa el espacio necesario, se "ajusta al contenido" Hola, soy otra etiqueta de span

Estilos css en HTML

Hay 3 formas de aniadir estilos:
      1 - en la etiqueta con el atributo style
      2 - en el head con la etiqueta style
      3- Linkeando a una hoja de estilos

Scripts en HTML

      Hay 2 formas:
      1 - Mediante la etiqueta script, de prefernecia van al fondo del body, antes del cierre
